Installation

Go to directory and run: 

$ npm install

To build run 

$ npm run build

In build/* folder the production ready files are generated

src/* folder contains actual code, where you can write edit/modify code.

config.js contains the configuration for the project and you can specify the api keys and server url for the api server here.

Setting up ads

public/* folder contains the files which would be included in the the build files. files like images, css, index.html etc are here.

You can specify the url fo the ads, currently I have used:

adcode.html
adcode2.html
adcode_header.html
adcode_small.html

You can edit the html directly and it will be displayed in the project.

Setting up csv

In the public/ folder, meta-descriptions.csv file is there, you can edit the file, a sample entry is already there, you can just copy and create a new row with matching data, it will be used as meta tags when given url opens.

Using pm2 for deployment

Install pm2 with 

$ sudo npm install -g pm2 
$ sudo npm install -g serve

go to the project folder, create a new file tclient.sh with following content 

  sudo serve -s /path/to/project/build -p 80

and run these commands where tclient.sh exists 

$ pm2 start tclient.sh
$ pm2 save
$ pm2 startup
